# CUDA OpenCV Caffe

Pushed to [alantrrs/cuda-opencv-caffe:nvidia](https://hub.docker.com/r/alantrrs/cuda-opencv-caffe/)

## Includes
- Cuda 7.5 
- OpenCV 3.0
- Caffe

## Use


### In your Dockerfile
```Dockerfile
FROM alantrrs/cuda-opencv-caffe:nvidia
```

### Pull
```
docker pull alantrrs/cuda-opencv-caffe:nvidia
```
